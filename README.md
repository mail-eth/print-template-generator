# Print Template Generator 🖼️

Website untuk generate template foto siap print. Masukkan ukuran foto, langsung jadi template yang bisa langsung dicetak!

## Fitur

- 📐 **Multiple Sizes** - Ukuran standar (2x3, 3x4, 4x6, 5x7, 8x10, A4, 3R, 4R, 5R, 8R)
- ✏️ **Custom Size** - Ukuran custom sesuai kebutuhan
- 🖨️ **Print Ready** - Langsung print dengan Ctrl+P atau tombol
- 📱 **Mobile Friendly** - Bisa pakai di HP

## Cara Pakai

1. Buka website
2. Pilih ukuran foto atau masukkan custom
3. Tentukan jumlah foto per sheet
4. Klik "Generate Template"
5. Klik "Print Sekarang"

## Ukuran yang Tersedia

| Kode | Inch | cm |
|------|------|-----|
| 2x3 | 2 x 3" | 5 x 7.6 cm |
| 3x4 | 3 x 4" | 7.6 x 10.2 cm |
| 4x6 | 4 x 6" | 10 x 15 cm |
| 5x7 | 5 x 7" | 12.7 x 17.8 cm |
| 8x10 | 8 x 10" | 20 x 25 cm |
| A4 | - | 21 x 29.7 cm |

## Development

Ini adalah static HTML file, bisa langsung dibuka di browser tanpa install apa pun.

```bash
# Buka langsung
open index.html

# Atau pakai local server
npx serve
```

## Tech Stack

- HTML5
- TailwindCSS (CDN)
- Vanilla JavaScript
